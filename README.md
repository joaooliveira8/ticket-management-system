# 💻 Helpdesk QA System

Projeto de Análise e Modelagem de Sistemas com foco em Qualidade de Software (QA), utilizando UML, levantamento de requisitos e criação de casos de teste.

---

## 📌 Escopo do Sistema

### 🖥️ O que o sistema faz

O Sistema de Gerenciamento de Chamados é uma aplicação que permite registrar, acompanhar e gerenciar solicitações de suporte técnico (chamados). A plataforma possibilita que usuários abram chamados descrevendo seus problemas, enquanto a equipe de suporte pode visualizar, atualizar o status, interagir com o solicitante e registrar a resolução.

O sistema também mantém um histórico completo das interações, garantindo rastreabilidade e organização no atendimento.

---

### 👥 Para quem é

Este sistema é voltado para empresas ou organizações que possuem uma equipe de suporte técnico, como:

* Empresas de tecnologia
* Setores de TI internos
* Provedores de serviço
* Help desks corporativos

Os principais usuários do sistema são:

* **Clientes/Usuários** → que solicitam suporte
* **Analistas de Suporte** → que tratam os chamados
* **Administradores** → que gerenciam o sistema

---

### ⚠️ Problema que resolve

Muitas empresas enfrentam dificuldades no controle de solicitações de suporte, como:

* Falta de organização dos atendimentos
* Perda de informações importantes
* Dificuldade em acompanhar o andamento dos chamados
* Falta de padronização no processo de suporte

O sistema resolve esses problemas ao centralizar todas as solicitações em uma única plataforma, permitindo melhor controle, rastreamento e eficiência no atendimento, além de proporcionar maior transparência para os usuários.
