# 🧪 Casos de Teste

## 📌 Caso de Teste 01 - Criar Chamado

**ID:** CT-001  
**Descrição:** Verificar se é possível criar um chamado com dados válidos  

**Passos:**
1. Acessar o sistema  
2. Preencher título e descrição  
3. Clicar em "Criar chamado"  

**Resultado Esperado:**
- O chamado deve ser criado com status "Aberto"  

---

## 📌 Caso de Teste 02 - Atualizar Status

**ID:** CT-002  
**Descrição:** Verificar se o analista pode atualizar o status do chamado  

**Passos:**
1. Acessar chamado existente  
2. Alterar status para "Em andamento"  

**Resultado Esperado:**
- O status deve ser atualizado corretamente
